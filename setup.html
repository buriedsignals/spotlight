How you'll run Spotlight with Claude Code

  1. Installer runs npm install -g @anthropic-ai/claude-code
  2. Installer clones the repo and symlinks AGENTS.md → CLAUDE.md so Claude Code picks up the Spotlight contract
  3. Installer adds a spotlight shell function to your ~/.zshrc (or ~/.bashrc)
  4. Open a new terminal and type spotlight — it opens Claude Code in the Spotlight repo with the skill loaded. Tell it: "Start a Spotlight investigation on <your lead>"

You can also use the Claude Desktop app: open the Spotlight repo folder, and Claude will see CLAUDE.md automatically.

Intelligence vault (Obsidian) REQUIRED

Spotlight archives verified findings into an Obsidian vault as structured knowledge — entities, methodology, tools — with wikilinks. Over time, the vault becomes your investigative memory.

Before you continue

  1. Install Obsidian 1.12+ from obsidian.md (if you don't have it already)
  2. Open Obsidian → Settings → General → Advanced → toggle Command Line Interface ON
  3. This adds the obsidian command to your PATH. Verify by running obsidian --version in Terminal.
  4. Create a new vault (or pick an existing one) and note its full path

Vault path is required.
